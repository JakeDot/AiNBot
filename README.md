# 🤖 AiNBot — AI-Native GitHub Repository Monitor with Claude Integration

A sophisticated P2P GitHub repository monitoring application built with Rust (libp2p), featuring advanced voting systems, risk management, and Claude AI integration with comprehensive security hardening via **[The CHEST](https://github.com/JakeDot/the-chest)**.

---

## 🏗️ Architecture

**Layers:**
- 🧠 **AI Layer** — Claude integration with vision support
- 🗳️ **Voting System** — Weighted voting, consensus resolution
- 📊 **Analytics** — Historical tracking, negotiation analytics
- 🛡️ **Security** — Built on [The CHEST](https://github.com/JakeDot/the-chest) fortress template
- 🌐 **P2P Network** — libp2p-based distributed monitoring

---

## 🔒 Security Foundation

This project is hardened with **[The CHEST](https://github.com/JakeDot/the-chest)** — enterprise-grade security template:

✅ Network security (firewall, TLS 1.3+)  
✅ Data encryption (AES-256-GCM)  
✅ Authentication (MFA, RBAC)  
✅ Audit & compliance (immutable logs)  
✅ Threat detection (real-time)  
✅ Supply chain security (dependency validation)  
✅ Incident response automation  

See [The CHEST README](https://github.com/JakeDot/the-chest#readme) for security framework details.

---

## 🚀 Quick Start

```bash
git clone https://github.com/JakeDot/AiNBot.git
cd AiNBot
npm install
npm run fortress:init
export FORTRESS_PROFILE=production
npm run build
npm start
```
